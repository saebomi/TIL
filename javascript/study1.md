## 2장 프로그램의 작성법과 실행법
### 2.4 프로그램 작성법
### 2.4.1 문자코드
- 유니코드 문자로 작성

### 2.4.2 대문자와 소문자
- 알파벳과 소문자를 구별함

### 2.4.3 토큰과 공백문자
- 토큰 : 프로그램을 구성하는 최소 단위
- 공백문자 : 토큰과 토큰 사이에 공백문자를 입력하여 구분. 공백문자 여러 개 입력 시 하나만 입력한 것으로 간주함

### 2.4.4 문장
- 문장 끝에는 반드시 세미콜론붙이기 (특별한 경우 제외)
- 복합문
: 문장 여러 개를 중괄호({})로 감싼 코드
